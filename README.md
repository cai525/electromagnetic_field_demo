# electromagnetic_field_demo

  该程序基于matlab的app演示，适用于辅助电磁场与电磁波学习理解。
 
 ## 环境
 编写使用了matlab2022。运行所需的matlab版本需要支持matlab APP。
 
 ## 程序结构为：
 ```mermaid
graph LR
A[main1] -->B[main2]
B --> C[app1]
B --> D[app2]
B --> E[app3]
```
