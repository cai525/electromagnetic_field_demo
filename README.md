# app_main
 - 适用于电磁场与电磁波以及微波技术的学习教学 
 
 该程序是基于matlab的app演示，适用于辅助电磁场与电磁波学习理解。
 
 matlab的版本需要支持matlab APP的运行。
 
 程序结构为：
 ```mermaid
graph LR
A[main1] -->B[main2]
B --> C[app1]
B --> D[app2]
```
